## UX Guidelines

### Core Principles
1. **Clarity** - Users understand immediately
2. **Consistency** - Predictable patterns throughout
3. **Feedback** - Response to user actions
4. **Efficiency** - Minimal steps to goal
5. **Accessibility** - Usable by everyone
6. **Aesthetics** - Beautiful, intentional design

### Animation Guidelines
- ✅ Use animations for guidance and feedback
- ✅ Keep animations quick (200-500ms)
- ✅ Provide motion preferences respect
- ❌ Don't animate loading states (too long)
- ❌ Don't use auto-play videos
- ❌ Excessive animations distract

### Accessibility (WCAG AA)
- [ ] Color contrast ≥ 4.5:1 for text
- [ ] Interactive elements ≥ 44x44px
- [ ] Form labels associated properly
- [ ] Images have alt text
- [ ] Keyboard navigable
- [ ] Focus visible always

### Dark Mode
- Invert colors appropriately
- Reduce pure white (use off-white)
- Increase opacity/brightness for text
- Test both modes thoroughly

## Design Decision Process

1. **Understand Context**
   - Industry? SaaS, E-commerce, Healthcare?
   - Audience? Age, tech-savviness?
   - Purpose? Convert, inform, entertain?

2. **Choose Style**
   - Match industry norms or disrupt intentionally
   - Align with brand personality
   - Consider target audience preferences

3. **Select Palette**
   - Choose primary brand color
   - Select complementary secondary
   - Use color structure (success, warning, danger)
   - Test contrast ratios

4. **Pick Typography**
   - Sans-serif for modern/clean
   - Serif for traditional/elegant
   - Match brand personality
   - Headline + body pair

5. **Design Layout**
   - Mobile-first approach
   - Follow proven landing patterns
   - Consistent spacing and alignment
   - Account for content

6. **Implement & Test**
   - Build in your framework
   - Test all breakpoints
   - Verify accessibility
   - Get user feedback
