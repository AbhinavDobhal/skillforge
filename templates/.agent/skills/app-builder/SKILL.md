# App Builder

## Core Guidance

- Start from the smallest end-to-end slice that proves the architecture.
- Connect product goals, technical scope, and validation strategy.
- Keep scaffolding aligned with the stack and deployment target.
- Prefer incremental delivery over large speculative setup.
