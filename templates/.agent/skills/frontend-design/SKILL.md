# Frontend Design

## Use This Skill To

- Choose layout and interaction patterns
- Design reusable component APIs
- Review accessibility and responsiveness
- Align product UX with implementation constraints

## Core Guidance

- Start with information hierarchy before styling decisions.
- Design mobile and keyboard behavior as first-class requirements.
- Keep component APIs predictable and easy to compose.
- Treat loading, empty, validation, and error states as part of the design.
