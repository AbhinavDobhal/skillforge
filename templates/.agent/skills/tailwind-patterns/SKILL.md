# Tailwind Patterns

## Core Guidance

- Use semantic component wrappers or variants when class strings grow beyond easy review.
- Prefer tokenized spacing, radius, and color scales over ad hoc values.
- Build responsive behavior intentionally instead of stacking utilities without hierarchy.
- Keep hover, focus, disabled, and dark-mode states consistent across components.
