# Web Design Guidelines

## Review Areas

- Visual hierarchy and spacing
- Contrast and readability
- Responsive behavior
- Motion and interaction feedback
- Keyboard and screen-reader support

## Minimum Bar

- Meet WCAG AA contrast for core text.
- Ensure focus styles are visible and intentional.
- Avoid layout shifts from images, fonts, and async content.
- Keep interactions obvious on touch and pointer devices.
